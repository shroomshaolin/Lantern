# Lantern
Lantern is a Sapphire plugin that creates a calm, focused space for reflective conversation, with a clean interface, polished transcript styling, and optional inner-thought visibility for deeper interaction.

## Features

- Polished conversation UI
- Clean transcript styling
- Thoughtful, distraction-free layout
- Optional inner-thought visibility
- Built for introspective and supportive interaction inside Sapphire

## Purpose

Lantern is designed to create a softer, more reflective conversational space than a standard chat interface, with an emphasis on clarity, comfort, and presence.

## Status

Active work in progress.

## Installation

Install Lantern as a Sapphire plugin in your Sapphire environment.

## Notes

This project is being developed and refined in a live Sapphire setup.
